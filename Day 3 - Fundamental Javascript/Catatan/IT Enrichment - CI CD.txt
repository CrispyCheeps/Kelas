CI/CD itu cingkatan dari continuous integration (CI) sm continuous delivery (CD)

Cakupan : langkah2 dr penggabungan kode hingga pengiriman ke produksi secara otomatis.

CI:
- praktik para pengembang itu nggabungin kode, kerjasama di dlm sebuah repository bersama; contohnya : git.
- Tujuan : Mendeteksi dan memperbaiki konflik pada siklus pengembangan. sehingga tim bisa segera ngatasi masalah klo ada isu muncul.

CD:
- Pkke semua proses pengiriman perangkat lunak dr pembangunan, pengujian, sampe tahap produksi.
